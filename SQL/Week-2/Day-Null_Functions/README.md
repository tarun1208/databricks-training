# 📚 SQL Concepts Covered

## ✅ IS NULL
Check whether a value is missing.



## ✅ IS NOT NULL
Check whether a value exists.


## ✅ IFNULL()
Replace NULL values with default values.



## ✅ COALESCE()
Return the first non-NULL value.


## ✅ NULLIF()
Convert values into NULL when conditions match.


## ✅ NULL Handling in Calculations
Prevent NULL issues during arithmetic operations.



## ✅ Divide-by-Zero Prevention
Using `NULLIF()` safely in calculations.
