## 📌 Project Overview

This repository contains my **Phase 2 SQL to PySpark Practice**, where I converted SQL queries into PySpark DataFrame operations using **Apache Spark**.

The objective of this phase was to bridge the gap between SQL and PySpark by solving real-world data engineering problems involving:

- Data Aggregation
- Joins
- Sorting
- Filtering
- SQL to PySpark Conversion
- Business Analytics Queries

All exercises were implemented using **Spark Playground sample datasets** and organized with professional project structure.
